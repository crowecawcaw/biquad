# Biquad Filter Designer

A simple web-based tool for designing and visualizing biquad digital filters.

## Features

- Configure filter parameters (type, frequency, Q, order)
- Visualize frequency response
- View filtered data in real-time
- Export filter coefficients

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
